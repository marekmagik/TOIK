Araucaria - Labels menu

Modify ownership (Ctrl + W)

Opens the Edit Owners dialog, where owners may be added to or deleted from selected nodes. Make sure you have selected the nodes for which the owners are to be modified before opening the Edit Owners dialog. In the dialog, you can add owners to the "Available owners" list on the left by typing the name of the owner in the text box and clicking "Add". Each owner has a Three-Letter-Acronym (TLA) which is used to label nodes in the main tree diagram.

Available owners may be deleted by selecting them in the list and then clicking "Delete". To add an one or more owners (multiple selections can be made by holding down the control key while selecting with the mouse), select in the Available Owners list those owners you wish to add to the selected nodes, then click the right-pointing arrow. The selected owners will appear in the "Owners assigned to propositions" list on the right. Similarly, to delete owners from the selected nodes, select in the right-hand list those owners you wish to delete and click the left-pointing arrow.


Hide/show owners (Ctrl + H)

Toggles the display of owners (as three-letter acronyms, or TLAs) on the main diagram (and also in the full text diagram). Owners appear as small green labels next to the nodes they own.


Modify evaluation (Ctrl + K)

Opens the Evaluation dialog, where evaluation tags can be added to selected diagram components. Make sure you have selected the nodes for which the evaluation is to be modified before opening the Evaluation dialog. In the dialog, you can add a new evaluation, or use one previously added in this analysis.


Hide/show evaluation (Ctrl + G)

Toggles the display of evaluation lables on the main diagram (and also in the full text diagram). Evaluations appear as small pink labels next to the nodes they evaluate.
