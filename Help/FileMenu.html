Araucaria - File menu

Open text file (Ctrl + T)

Reads text from a .txt file and displays it in the blue panel on the left. This will erase any existing tree, so you may wish to save this using 'save argument' first.


Open argument (Ctrl + O)

Reads in the scheme set, text and tree structure from an AML file, and displays them. Further editing is allowed on trees.


Save argument (Ctrl + S)

Saves the current scheme set, text and tree structure as an AML file. This file can be read in by Araucaria, but should also be useable by any other program that defines meanings for the XML tags defined by the AML DTD.


Save as

Saves the argument analysis under a new name .


Save diagram (Ctrl + D)

Saves the current image of the diagram (minus any unattached premises at the bottom of the diagram) as a JPEG file.


Close all

Close the current files


Properties

Show summary statistics concerning the current argument analysis, and provide access to the source and comments fields.


Preferences

Change system settings, including colours, file paths and AraucariaDB database connection properties.


Exit

Quits the program. You will be asked if you want to save any modified diagrams first.
