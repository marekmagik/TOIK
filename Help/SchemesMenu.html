Araucaria - Schemes menu

Select scheme (Ctrl + E)

Allows a sub-argument within the tree to be labelled with an argument scheme. Select all edges (not nodes) that are to belong to the scheme, then select this menu item. A dialog is displayed, showing the types of scheme available. The required premises and conclusion, along with the critical questions, for each scheme are shown, as are the premises and conclusion from the selected items within the diagram.


Add/edit scheme (Ctrl + I)

Allows argument schemes to be edited, added or deleted. This will bring up a dialog showing the current set of argument schemes that are loaded into the program. You may select an existing scheme for editing, create a new scheme, or delete an existing scheme. Within each scheme, premises and critical questions may be edited in a similar way.


Save schemeset (Ctrl + V)

Saves the current schemeset as an AML file, but with a .scm extension (to distinguish it from saved arguments, which have an .aml extension).


Open schemeset (Ctrl + P)

Reads a schemeset from an .scm file, overwriting any existing schemeset.
